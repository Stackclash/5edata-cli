5edata-cli
=================

A CLI tool to convert 5etools JSON and D&D Beyond character data to markdown files.

# Usage
<!-- usage -->
```sh-session
$ npm install -g @stackclash/5edata-cli
$ 5edata COMMAND
running command...
$ 5edata (--version)
@stackclash/5edata-cli/1.0.0 linux-x64 node-v24.8.0
$ 5edata --help [COMMAND]
USAGE
  $ 5edata COMMAND
...
```
