`5edata help`
=============

Display help for 5edata.

* [`5edata help [COMMAND]`](#5edata-help-command)

## `5edata help [COMMAND]`

Display help for 5edata.

```
USAGE
  $ 5edata help [COMMAND...] [-n]

ARGUMENTS
  COMMAND...  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for 5edata.
```
